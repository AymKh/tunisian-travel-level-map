# Tunisian Travel Level Map

Mark the places you've visited and see which other more you wanna go to!

| Level | Description |
| ----- | ----------- |
| 0 | Never been there |
| 1 | Passed here |
| 2 | Alighted here |
| 3 | Visited here |
| 4 | Stayed there |
| 5 | Lived there |
